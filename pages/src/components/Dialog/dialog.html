Dialog

Content

HTML


    <div class="d1" data-title="Title">Content</div>
    <button class="b1">normal dialog</button>
    <button class="b2">alert dialog</button>
    <button class="b3">confirm dialog</button>

JavaScript


    $('button').bizButton({
        theme: 'blue-gray'
    });
    $('.d1').bizDialog({
        buttons: [
            {
                text: 'OK',
                onClick: function() {
                    this.close();
                }
            },
            {
                text: 'Cancel',
                theme: 'gray',
                onClick: function() {
                    this.close();
                }
            }
        ]
    });
    $('.b1').click(function() {
        $('.d1').bizDialog('open');
    });
    $('.b2').click(function() {
        bizui.alert({
            title: 'Alert',
            content: 'Content',
            theme: 'red'
        });
    });
    $('.b3').click(function() {
        bizui.confirm({
            title: 'Confirm',
            content: 'Content',
            theme: 'green'
        });
    });

Options

参数名称 参数类型 描述
buttons Array 底部按钮的 option 数组（多一个 onClick 属性），默认 [ ]
customClass String 自定义 CSS class
destroyOnClose Boolean 关闭时销毁对话框，默认 false
height Number 对话框高度，默认取填充内容后的高度，最小 240px
position String 定位方式（fixed|absolute），默认 'fixed'
theme String 主题
title String 对话框标题，默认取目标元素的 data-title 属性
width Number 对话框宽度，最小 480px
onBeforeClose Function 关闭前回调，返回 false 则不关闭
zIndex Number 对话框 z-index

Methods

close()

关闭对话框

destroy()

销毁

open()

打开对话框

title([String title])

获取/设置标题

updateButtons(Object buttonOption)

更新按钮

bizui.alert(Object options)

提示对话框

参数名称 参数类型 描述
content String 内容，默认 ''
okText String 确认文字，默认 '确定'
theme String 主题
title String 标题，默认 ''

bizui.confirm(Object options)

确认对话框

参数名称 参数类型 描述
content String 内容，默认 ''
cancelText String 取消文字，默认 '取消'
okText String 确认文字，默认 '确定'
theme String 主题
title String 标题，默认 ''
onOK function 确认回调，返回 false 则不关闭
