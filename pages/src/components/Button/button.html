Options

参数名称 参数类型 描述
theme String 主题
size String 尺寸（small|large）
icon String 图标名称或 codepoint
label String 文字
disabled Boolean 是否禁用

Methods

destroy()

销毁

disable()

禁用

enable()

激活

兼容性问题

对于 IE 浏览器，icon 字段要使用 codepoints，例如：


    $('.done').bizButton({icon: '&#xE876;'});
